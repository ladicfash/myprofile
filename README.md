## Deploy to GitHub Pages

See the [GitHub Pages setup steps](#) below, or follow the quick version:

1. Go to your repo → **Settings** → **Pages**
2. Under *Source*, select **Deploy from a branch**
3. Choose **main** branch, **/ (root)** folder
4. Click **Save**
5. Your site will be live at `https://ladicfash.github.io/myprofile` within ~60 seconds
